### To use just test in node:
  1.  Type in npm install jest
  2.  type in npm install -- save dev jest 
  3.  the command in node is npm test

  4.  Included in the package.json file:

  {
  "scripts": {
    "test": "jest"
  },
  "dependencies": {
    "jest": "^27.0.4"
  },
  "env": {
    "jest": true
  }

  }
